# Tools

> 想要教给人们一种新的思维方式，就不要刻意去教，而应当给他们一种工具，通过使用工具培养新的思维模型。 
> 
> source: 《第五项修炼》

## Why?


人之于世，必有所凭借，即工具。善用工具者，四两拨千斤。目前思考的问题：

* 工具箱是否**完备**？高效地完成生活的各个环节。
* 工具是否**足够优**？不必最优，只要满足自己的需求即可。

如何选择工具？
